    • Using GitHub

    The main purpose of Github, is to store your code, you can see it as a library, where you put your code and check it whenever you want.

    There are 5 main commands on git :

  • git add (To add the code you made in your repo)
  • git commit -m (To "save" the changes you made an give them a name, by adding the "-m")
  • git push (To send your code on the repo)
  • git clone (To clone a repo from its url. You can change his name by giving him a name after the clone)
  • git pull (To get the last update on your branch)
  • git checkout -b (To create a new branch and work on it)
